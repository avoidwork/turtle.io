Method handle

Handles the request

Parameters:

  • req must be an Object.
    (HTTP(S) request Object)

  • res must be an Object.
    (HTTP(S) response Object)

  • path must be a String.
    (File path)

  • url must be a String.
    (Requested URL)

  • dir must be a Boolean.
    (true is path is a directory)

  • stat must be an Object.
    (fs.Stat Object)

Returns an Object
(TurtleIO instance)
